# 마크다운 문법 정리

---

## 헤더 / Header
- '#' 의 숫자만큼 <h#> Tag 라고 이해


**예시**

```
# 헤더
## 헤더
### 헤더
```

**결과**
# 헤더

## 헤더

### 헤더

---


## 스타일 Text


이텔릭체는 *별표(asterisks)* 혹은 _언더바(underscore)_를 사용


두껍게는 **별표(asterisks)** 혹은 __언더바(underscore)__를 사용


**_이텔릭체_와 두껍게**를 같이 사용


취소선은 ~~물결표시(tilde)~~를 사용


<u>밑줄</u>은 `<u></u>`를 사용
